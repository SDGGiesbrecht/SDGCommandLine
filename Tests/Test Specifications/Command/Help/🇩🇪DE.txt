$ werkzeug execute help •no‐colour

werkzeug ausführen führt eine erfolgreiche Ausführung vor.

Es gibt ein paar Optionen.

Unterbefehle

hilfe zeigt Gebrauchsinformationen an.

Optionen

•farbe [Farbe] Eine Farbe für den Text.
•informell verwendet eine informelle Begrüßung.
•ohne‐farben Blendet Farben aus der Ausgabe aus.
•path [Pfad] A directory to run in.
•sprache [Spracheinstellung] Eine Sprache anstelle deren zu verwenden, die in den Einstellungen angegeben ist.
•unsatisfiable [unsatisfiable] An option that always fails to parse.
•version‐verwenden [Version] Versucht die angegebene Version herunterzuladen und vorübergehend anstelle deren zu verwenden, die installiert ist.
•wiederholungen [1–5] Die Anzahl der Wiederholungen.
•zeichenkette [Zeichenkette] Text zum Zeigen.

Argumentarte

[1–5] Eine ganze Zahl zwischen 1 und 5 einschließlich.
[Farbe] „blau“, „grün“ oder „rot“.
[Pfad] Ein Pfadname. Die Form „/...“ gibt einen vollständigen Pfad an. Die Form „~/...“ gibt einen relativen Pfad an, ausgehend von dem Benutzerverzeichnis. Alles andere gilt als relativer Pfad, ausgehened vom aktuellen Arbeitsverzeichnis.
[Spracheinstellung] Eine Liste IETF Sprachbezeichnungen oder SDGCornerstone‐Sprachsymbole. Doppelpunkte geben die Ersatzreihenfolge an. Kommata geben an, dass mehrere Sprachen verwendet werden sollen. Beispiele: „de-DE“ oder „🇩🇪DE“ → Deutsch aus Deutschland, “bar,de;fr” → beide Bairisch und Deutsch, sonst Französisch
[Version] Eine Versionsnummer oder „entwicklung“.
[Zeichenkette] Eine beliebige Zeichenkette.
[unsatisfiable] An argument type that always fails to parse.
